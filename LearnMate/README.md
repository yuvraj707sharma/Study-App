# LearnMate

LearnMate is your personal learning assistant designed to help you study based on your unique learning style! Whether you're a visual, auditory, kinesthetic, or reading/writing learner, LearnMate adapts to you with tailored study tools.

## Features
- **Visual Learners:** Mind map generator, diagram creator, color-coded notes
- **Auditory Learners:** Text-to-speech summaries, audio explanations
- **Kinesthetic Learners:** Interactive quizzes, gamified exercises
- **Reading/Writing Learners:** Smart note-taking, automated summaries

## Tech Stack
- React (frontend)
- Node.js + Express (backend)
- TypeScript (where possible)

## Getting Started
1. Install dependencies: `npm install` (in both frontend and backend folders)
2. Start backend: `npm run start` (in backend)
3. Start frontend: `npm start` (in frontend)
